# *pokerparty!*
## by Ryder Pham and Krishna Katakota


### Instructions for running locally
* clone repo
* clone [backend repo](https://github.com/krishnakatakota/pokerparty-ws-server) and run that according to its readme's instructions
* run `npm install`
* run `ng serve`
